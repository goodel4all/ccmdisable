# 🌟 AI 학습 친구 (AI Learning Friend)

지적장애 학생을 위한 접근성이 높고 안전한 AI 학습 도우미입니다.

## 🚀 주요 기능
- **친절한 선생님 페르소나**: 학생의 눈높이에 맞춘 쉽고 긍정적인 말하기
- **접근성 중심 UI**: 큰 글씨와 명확한 아이콘으로 구성된 인터페이스
- **학습 모드**: 읽기, 수학, 생활 기술 등 다양한 주제 지원
- **안전한 환경**: 구글 Gemini API의 세이프티 설정을 통한 유해 콘텐츠 차단

## 🛠️ 설치 및 실행 방법

1. **파이썬 패키지 설치**
   ```bash
   pip install -r requirements.txt
   ```

2. **애플리케이션 실행**
   ```bash
   streamlit run app.py
   ```

3. **API 키 설정**
   - 실행 후 웹 브라우저의 사이드바에서 [Google AI Studio](https://aistudio.google.com/app/apikey)에서 발급받은 API 키를 입력하세요.

## 📝 기술 스택
- **Language**: Python 3.10+
- **Framework**: Streamlit
- **AI Model**: Google Gemini 1.5 Flash
